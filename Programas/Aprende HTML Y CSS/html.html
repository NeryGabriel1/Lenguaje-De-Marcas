¿Que es Html?

HTML es el acrónimo de HyperText Markup Language. Es un lenguaje de marcado que se utiliza para crear páginas web y aplicaciones web. Básicamente, HTML se compone de etiquetas que se utilizan para definir la estructura y el contenido de una página web. Cada etiqueta tiene un propósito específico y se utiliza para definir diferentes tipos de elementos, como encabezados, párrafos, imágenes, enlaces, formularios, entre otros.


HTML es un lenguaje muy fácil de aprender y es la base de la mayoría de las páginas web que se ven en Internet hoy en día.

Estructura de una página HTML

La estructura básica de una página HTML consta de los siguientes elementos:
<!DOCTYPE html>: Este es el doctype, y su presencia es necesaria para que el navegador interprete correctamente el documento como HTML5.
<html>: Este es el elemento raíz del documento HTML.
<head>: Este elemento contiene metadatos del documento, que no se muestran en la página web, pero son útiles para los navegadores y los motores de búsqueda. Algunos de los elementos que puedes encontrar dentro de <head> incluyen:
<title>: Define el título del documento, que se muestra en la barra de título del navegador o en la pestaña del navegador.
<meta>: Se utiliza para especificar el conjunto de caracteres, la descripción de la página, las palabras clave, el autor del documento y la vista de puerto.
<link>: Se utiliza para enlazar hojas de estilo externas.
<style>: Se utiliza para escribir CSS interno.
<script>: Se utiliza para enlazar y escribir JavaScript.
<body>: Este elemento contiene el contenido principal que se muestra a los usuarios cuando visitan tu página web. Puede contener texto, imágenes, tablas, enlaces, formularios, etc.
Dentro del <body>, puedes tener una variedad de otros elementos HTML, como <h1> a <h6> para encabezados, <p> para párrafos, <a> para enlaces, <img> para imágenes, <div> y <span> para agrupar contenido, <ul> y <ol> para listas, <table> para tablas, y muchos más.
Aquí tienes un ejemplo de cómo se ve una estructura HTML básica:


<!DOCTYPE html>
<html>
  <head>
    <title>Título de la página</title>
  </head>
  <body>
   <h1>Este es un encabezado</h1>
    <p>Este es un párrafo.</p>
  </body>
</html>

Etiquetas semanticas

Las etiquetas semánticas en HTML son aquellas que proporcionan información sobre el tipo de contenido que contienen, lo que ayuda a los motores de búsqueda a entender mejor la estructura y el contenido de la página web. Aquí tienes algunos ejemplos de etiquetas semánticas en HTML:

Etiqueta HTML Descripción Ejemplo
<header> Representa un grupo de ayudas introductorias o de navegación. También puede contener logos, formulario de búsqueda o tabla de contenidos.
Contenido del encabezado
<nav> Sección de una página que enlaza a otras páginas o partes de la misma. Una sección con links de navegación.
Contenido de la navegación
<aside> Sección de una página que contiene contenido relacionado tangencialmente al de su alrededor. Por lo general se utilizan como barras laterales.
Contenido del aside
<main> Representa el contenido predominante de la página. Una página debe tener solo un tag
.
Contenido principal
<article> Representa una sección de contenido que puede ser distribuido de forma independiente. Puede ser un post en un foro, un articulo de periódico, una entrada de un blog, un comentario, etc.
Contenido del artículo
<section> Representa a una sección genérica de una página. Agrupa contenidos que tienen una relación temática entre sí.
Contenido de la sección
<footer> Representa un pie de página para el elemento que lo contiene. Generalmente contiene información acerca de quién lo escribió, enlaces a documentos relacionados, datos de derechos de autor o similares.

    Elementos de Lineas

    Un elemento de línea es un elemento que no comienza con una nueva línea cuando se representa en el navegador. Estos elementos solo ocupan el espacio necesario para el contenido que contienen, en lugar de ocupar toda la línea. No puedes establecer dimensiones (ancho y alto) en los elementos de línea.     Atributos HTML

    Los atributos en HTML son valores adicionales que configuran los elementos o ajustan su comportamiento. Cada elemento de HTML tiene sus propios atributos, que se utilizan para definir su apariencia, comportamiento y funcionalidad.Aquí hay una lista de algunos de los atributos más comunes en HTML:

    ATRIBUTOS HTML Descripción Ejemplo
    <class> Este atributo especifica una o más clases para un elemento HTML. Las clases se utilizan para seleccionar elementos con selectores CSS específicos. <div class="miClase"></div>
    <id> Este atributo especifica un identificador único para un elemento HTML. El valor del atributo id debe ser único dentro del documento HTML. <div id="miID"></div>
    <style> Este atributo se utiliza para aplicar estilos CSS en línea a un elemento HTML. <p style="color:red">Este es un párrafo rojo.</p>
    <lang> Este atributo declara el idioma de la página web o solo una parte de la página web. Se utiliza para ayudar a los motores de búsqueda y navegadores. <html lang="es"></html>
    <src> Este atributo especifica la URL de un recurso externo. Se utiliza comúnmente en las etiquetas <img>, <script> y <iframe>. <img src="miImagen.jpg">
    <href> Este atributo especifica la URL de la página a la que enlaza un enlace. Se utiliza en la etiqueta <a>. <a href="https://www.ejemplo.com">Visita nuestra página web</a>
